# Notes-app

Notes App build with MongoDB, ExpressJS, ReactJS, and NodeJS


# Installation
1. Use the package manager [npm](https://www.npmjs.com/) to install modules.
  ```bash
npm install
```
then do
  ```bash
npm run install
```
2. After that, run the whole app with
  ```bash
npm run gas
```
